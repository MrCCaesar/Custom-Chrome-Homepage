# Custom Chrome Homepage

一个功能丰富的 Chrome 新标签页扩展，替换默认新标签页。

## 功能

- 🔍 **多搜索引擎搜索栏** — 支持 Google / Bing / 百度 / DuckDuckGo / GitHub，支持自定义 URL 搜索引擎
- ✏️ **编辑/显示双模式** — 显示模式清爽浏览，编辑模式自由定制
- 🖱️ **拖拽自由定位** — 编辑模式下搜索栏和分组可拖拽到任意位置
- 🖼️ **自定义背景图片** — 全局背景 + 每个分组独立背景图
- 📑 **分组书签管理** — 网站链接按分组管理，分组可独立命名，自动获取网站图标
- 📐 **图标尺寸连续调节** — 32px-96px 滑块无极调节
- ☁️ **跨设备自动同步** — 设置和书签通过 Chrome 账号自动跨设备同步（`chrome.storage.sync`）
- 📤 **导出/导入配置** — 一键导出全部配置为 JSON 备份，支持导入恢复

## 安装方法

1. 生成图标（需要 Python + Pillow）：
   ```bash
   pip install Pillow
   python generate_icons.py
   ```

   或手动将任意 16×16、48×48、128×128 的 PNG 图片放入 `icons/` 目录。

2. 打开 Chrome → `chrome://extensions/` → 开启「开发者模式」

3. 点击「加载已解压的扩展程序」→ 选择本项目文件夹

4. 打开新标签页即可使用

## 使用说明

### 模式切换

| 操作 | 方式 |
|------|------|
| 切换到编辑模式 | 点击右上角 ✎ 按钮 |
| 切换到显示模式 | 点击右上角 👁 按钮（紫色发光状态下） |

### 显示模式

- 搜索栏输入关键词 → 回车搜索
- 点击链接卡片 → 打开网站
- 界面清爽无编辑元素

### 编辑模式

| 操作 | 方式 |
|------|------|
| 打开全局设置 | 点击右下角 ⚙️ 按钮 |
| 添加分组 | 页面底部「+ 添加分组」 |
| 重命名分组 | 点击分组标题 |
| 删除分组 | 分组标题旁 ✕ 按钮 |
| 添加链接 | 分组内「+ 添加链接」 |
| 删除链接 | 鼠标悬停链接 → 点击红 ✕ |
| 设置分组背景图 | 分组标题旁 🖼 按钮 |
| 拖拽移动搜索栏 | 按住左边 ⋮ 手柄拖动 |
| 拖拽移动分组 | 按住左上角 ⋮⋮ 手柄拖动 |
| 恢复默认位置 | 设置面板 → 🔄 按钮 |

### 设置面板

| 设置项 | 说明 |
|------|------|
| 搜索引擎 | 内置引擎 + 自定义 URL 模板 |
| 搜索栏初始 Y 位置 | 滑块调节，拖拽后自动更新 |
| 全局背景图片 | 上传/清除 |
| 图标尺寸 | 32-96px 连续滑块 |
| 数据同步 | 自动通过 Chrome 账号跨设备同步 |
| 导出/导入配置 | JSON 备份与恢复 |

## 数据存储架构

```
chrome.storage.sync（跨设备自动同步）
├── syncSettings_v2: 搜索引擎、位置、图标大小、编辑模式
└── syncBookmarks_v2: 分组名、链接（名/URL/图标）

chrome.storage.local（仅本地）
└── localBg_v2: 全局背景图 + 分组背景图 (Data URL)
```
